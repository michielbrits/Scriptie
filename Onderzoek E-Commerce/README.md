# Tea commerce

### Beschrijving
Tea Commerce is een lichte e-commerce oplossing voor Umbraco – Het is makkelijk om te implementeren en komt met een simpele shop interface voor de shop eigenaar. 
Tea Commerce biedt je de mogelijkheid om een Umbraco e-commerce oplossing te implementeren door alleen gebruik te maken van Razor/XSLT, HTML, CSS en javascript. Geen kennis van .NET is gevraagd.
### Werking
Je kan lijsten van een type van product aanmaken, vervolgens kan je daaronder producten aanmaken.
Je kan varianten aanmaken, bijvoorbeeld schoenmaat. Hierna kan je per product in stellen welke variaties je op de producten kan zetten.
### Voordelen
Integreert heel makkelijk met de umbraco structuur. Het is ook heel makkelijk in gebruik als je de umbraco structuur eenmaal gewoon bent. 
Het is ook heel simpel in gebruik, je maakt gewoon pagina’s aan in Umbraco.
De licentie is 1 keer te betalen en blijft geldig. ( dus niet per jaar )
### Nadelen
Als je in productie gaat is een licentie van 1300 euro vereist. Deze licentie blijft geldig.
De opbouw van deze structuur is vrij simpel, je kan bijvoorbeeld 1 product maar in 1 categorie steken dit is een groot nadeel ten opzichte van de andere E-commerce systemen.
 
# Ucommerce
### Beschrijving
uCommerce is een uitgebreide e-commerce package
### Voordelen
Uitgebreide documentatie
Gratis voor kleine webshops
Volledig aanpasbaar
Custom checkout flow
Standaard meer dan 25 betaalmogelijkheden en je kan eigen betaalmogelijkheden definiëren
### Nadelen
Sommige features zijn uitgeschakeld in de gratis versie
Producten in aparte sectie
Zeer uitgebreid -> verwarrende UI
Interface loopt af en toe vast (misschien beter in productie ?)
Eigen UI -> niet de gebruiksvriendelijke features waarover Umbraco beschikt

 

# Merchello

### Beschrijving

De meest gebruikte gratis E-commerce oplossing voor Umbraco.
Het focust op het het maken van E-commerce shops voor je klanten en integreert zonder problemen in de Umbraco Backend. 

### Voordelen
Merchello is aanpasbaar, je kan het makkelijk aanpassen aan de noden van de klant. De sterke punten zijn aanpasbaar, schaalbaar en uitbreidbaar. Het grootste voordeel ten opzichte van Ucommerce en Tea commerce is dat Merchello volledig gratis in gebruik is, er hoeft dus ook geen dure licentie van ettelijke duizenden euros aangekocht te worden. Merchello is open-source.
Merchello integreert ook makkelijk in Umbraco. Het voelt ook aan als umbraco, klanten en editors kunnen heel makkelijk de omgeving managen via de Umbraco backend omgeving.
Nadelen
Merchello maakt een aparte tab aan in Umbraco. Het aanmaken van producten is minder intuïtief dan bijvoorbeeld in Tea Commerce. (Zeker voor iemand met een niet-development achtergrond zal dit omslachtiger zijn)


#Conclusie

Nadat we deze packages uitvoerig getest en vergeleken hebben heeft Sander beslist dat we verder gaan met de Merchello packaging. Met als grootste doorslaggevende factor dat Merchello open-source is.
