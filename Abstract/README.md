## Abstract
<!--Het abstract is een samenvatting van je totale bachelorproef, inclusief reeds gekende resultaten-->
Voor het bedrijg ABC-Groep hebben wij een nieuwe website ontwikkeld. De oude website was verouderd en ook niet echt geoptimalisseerd voor de weergave op een desktp op laptop. We zijn gestart van een template: bootstrap versa minimal business template. We hebben stukken code uit deze template gebruikt voor het opbouwen van de website. Sander (onze projectmanager) had ons een word gestuurd waarin stond wat er per pagina voorzien moest worden. De pagina's die geimplementeerd zijn geweest zijn: homepagina, technolgie, diensten, over ons, projecten, vacatures, Contact. Bepaalde pagina's hadden ook nog child pagina's. Door middel van een dropdown in de navigatie van de website kan je dan naar deze child pages navigeren.
### De essentie
De essentie van wat wij gedaan hebben is het schrijven van Razor code en het aanmaken van pagina's in het backend umbraco gedeelte. De razor code die wij geschreven heeft als functie de content van de website bouwer te laden van de backend en rendert het naar de frontend op de juiste plaats(en). Umbraco is heel aanpasbaar op de umbraco backend office maak je als developper een Document type aan. In deze document type kan je dan verschillende properties instellen. Zo kan je bijvoorbeeld een textstring gebruiken handig voor een naam van een link in te stellen. Een richetext editor is een meer uitgebreide tool. Dit kan bijvoorbeeld gebruikt worden voor een vacature in te stellen of een nieuws pagina. Het voordeel van een richtext editor is dat je ook opmaak aan een text kan meegeven die je hierin typt. Dit is maar 1 van de vele properties die Umbraco te bieden heeft, maar dient louter om ons verhaal duidelijker te maken.
